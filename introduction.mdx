## What you can do with Geo Web IW

Geo Web IW is built around three core capabilities:

**Maps & Layers** — Create interactive, multi-layer maps from your own data and external sources. Style features by attribute, toggle layers dynamically, and embed maps anywhere.

**Intelligent Workflows** — Build automated processes that respond to spatial events: geofence crossings, proximity triggers, scheduled data refreshes, and more. Connect maps to your existing tools via webhooks and the REST API.

**Spatial Data Management** — Import, query, and export geospatial data in standard formats (GeoJSON, Shapefile, CSV with coordinates, KML). Run spatial queries like intersect, buffer, and nearest-neighbor without writing SQL.

## Getting started

<Steps>
  <Step title="Create your account">
    Sign up at [app.geowebhq.com](https://app.geowebhq.com). Free trials include all features for 14 days — no credit card required.
  </Step>
  <Step title="Import your data">
    Upload a GeoJSON, Shapefile, or CSV file to create your first data layer. Geo Web IW automatically detects coordinates and renders your data on a map.
  </Step>
  <Step title="Build your first map">
    Add layers, configure styles, and set up filters. Publish your map with a shareable link or embed it in your application.
  </Step>
  <Step title="Connect workflows">
    Add a workflow to react to spatial events — send a notification when an asset enters a zone, sync data to an external system, or trigger a report on a schedule.
  </Step>
</Steps>
